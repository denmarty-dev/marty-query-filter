## [1.0.0] - 2026-05-01

### Added

- Initial release of `MartyQueryFilter`
- `QueryFilter` contract for reusable Eloquent filters
- `FilterPipeline` for filter registry resolution and execution
- Built-in `SearchFilter` with model and relation search support
- Laravel service provider auto-discovery support
- Pest test suite for pipeline and search behavior
